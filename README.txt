# Gin Example

This is a sample project, that uses gin to create a webapp.


## Features

- Support for a configuration file written in json, yaml, toml or env to configure the server;
- Proper handling of signals in order to control the server;
- Support for session handled by a redis server;
- Support for a postgresql database.
